#  Google Homepage Clone

This project is a simple clone of the **Google Homepage**, created as part of the [Angela Yu Web Development Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/).

Your site is live at https://nawfal-developer.github.io/google-homepage-clone/


##  Project Description



The Google Homepage Clone replicates the layout and design of Google’s famous homepage using **HTML** and **CSS**. The project focuses on creating a clean, minimalistic layout with a centered search box, logo, and footer.

##  Features

- layout with a centered search box
- Custom Google logo placeholder
- Styled search input and button
- Footer with links, styled using CSS

##  Technologies Used

- HTML5
- CSS3 (with Flexbox for centering)
  
##  What I Learned

- Creating a clean and minimal webpage design
- Using Flexbox for centering elements
- Styling basic form elements (input, button)
